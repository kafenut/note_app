0.git介绍
git是一个完全免费的分布式文件控制系统，其功能非常强大。
git的主要功能是管理文件版本库，可追踪文件每个版本的改动，能够多人协作完成同一份文档，并记录下每个人的改动，还能够指定远程仓库，对文件进行远端备份。
而GitHub的出现更是让git风靡了全球，GitHub给开源用户们提供了免费的git远程仓库服务，即你可以在git中指定GitHub为远端仓库，进行云备份，并且能与全世界的人协作完成你的开源项目。在中国，你也可以用码云作为git的远端仓库。

不过有一点必须提一下，所有文件版本控制系统，都只能追踪纯文本文件的改动（如第八行添加了个apple），无法追踪二进制文件，如视频，图片文件的改动。
不幸的是，word文档是二进制格式，所以我们没法用git管理dox文档。程序代码，txt这些纯文本格式的文件才是git的主要目标。

还有一点，windows自带的记事本会在保存文档时会使用UTF-8 with BOM的方式保存（vs也会）。确切些，记事本会在保存文档时，在文档前加上0xEFBBBF（16进制）的一共三个字节的前缀。这个前缀不仅没有任何意义，反而会引发许多神秘的问题（比如说我的这个网页用UTF-8 with BOM保存时，在网页顶会空出一行）。
为了代码文本的泛用性，一般情况下我们都推荐使用UTF-8的格式保存文档，而为了防止window的UTF-8 with BOM格式前缀带来的问题，我们可以使用其他的文本编辑软件。我用的是notepad++，是一个完全免费的开源文本编辑软件。

1.安装git
直接在GitHub官网下载就好：
https://git-scm.com/downloads
在命令行中输入git即可查看git是否安装成功。window安装成功后会有一个叫git bash的程序，点开后可在linux命令行环境里输入命令。

2.基础配置
在命令行里输入：
git config --global user.name "Your Nickname"
git config --global user.email "Your email"
之后我们就配置好了本地git用户的名称和email，其用于git分布式系统的最基础的用户认证。--global选项会让所有在这个机器上的git仓库都使用这个email与用户名。
